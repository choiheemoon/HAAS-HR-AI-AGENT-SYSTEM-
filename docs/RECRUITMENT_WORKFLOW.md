# 채용관리 Workflow 가이드

## 📋 전체 채용 프로세스 개요

### 1단계: 채용 요청 및 공고 작성

#### 1.1 채용 요청 생성
- **담당자**: 부서장/팀장
- **액션**: 
  - 채용 공고 작성 모달 열기
  - 필수 정보 입력 (제목, 부서, 직책, 근무지, 급여 범위 등)
  - 자격 요건 및 직무 설명 작성
- **상태**: `DRAFT` (초안)

#### 1.2 AI 기반 공고 최적화 (선택)
- **시스템**: AI Agent 자동 실행
- **액션**:
  - 자격 요건을 기반으로 채용 공고 자동 생성
  - 키워드 최적화 및 매력적인 문구 제안
- **상태**: `DRAFT` → `DRAFT` (개선됨)

#### 1.3 채용 공고 검토 및 승인
- **담당자**: HR 담당자/인사팀
- **액션**:
  - 작성된 공고 검토
  - 필요시 수정 요청
  - 승인 버튼 클릭
- **상태**: `DRAFT` → `PENDING_APPROVAL` → `APPROVED`

---

### 2단계: 채용 공고 배포 및 지원자 모집

#### 2.1 채용 공고 배포
- **담당자**: HR 담당자
- **액션**:
  - 배포할 채용 사이트 선택 (사람인, 잡코리아, 인크루트 등)
  - 배포 버튼 클릭
- **상태**: `APPROVED` → `PUBLISHED`
- **자동화**:
  - 각 채용 사이트 API 연동 (향후 구현)
  - 공고 자동 등록

#### 2.2 AI 소싱 (자동 후보자 검색)
- **시스템**: AI Sourcing Agent
- **액션**:
  - LinkedIn, GitHub 등 외부 플랫폼 자동 검색
  - 프로필 분석 및 매칭 점수 계산
  - 적합한 후보자 자동 추천
- **결과**: 후보자 목록 및 AI 매칭 점수 제공

---

### 3단계: 지원서 접수 및 이력서 파싱

#### 3.1 지원서 접수
- **지원자**: 외부 지원자
- **액션**:
  - 채용 공고 페이지에서 지원
  - 이력서 파일 업로드 (PDF, DOCX, TXT)
  - 자기소개서 작성
- **상태**: `APPLIED` (지원)

#### 3.2 이력서 자동 파싱 (ATS)
- **시스템**: Resume Parser + AI Agent
- **자동 처리**:
  - 이력서 파일에서 텍스트 추출
  - 구조화된 데이터로 변환:
    - 개인정보 (이름, 이메일, 전화번호)
    - 경력 정보 (경력 연수, 회사명, 직책)
    - 학력 정보
    - 기술 스택 (Skills)
    - 자격증
  - AI 매칭 점수 계산 (채용 공고 요구사항 대비)
- **상태**: `APPLIED` → `SCREENING` (서류심사)

---

### 4단계: 서류 심사 및 평가

#### 4.1 1차 서류 심사
- **담당자**: HR 담당자/채용 담당자
- **액션**:
  - 지원자 목록 확인
  - AI 매칭 점수 및 파싱된 정보 검토
  - 이력서 다운로드 및 상세 검토
  - 평가 점수 입력 (0-100점)
  - 평가 노트 작성
- **결정**:
  - **합격**: 면접 일정 조율
  - **불합격**: 거절 메시지 전송
- **상태**: `SCREENING` → `INTERVIEW_SCHEDULED` 또는 `REJECTED`

#### 4.2 자동 필터링 (선택)
- **시스템**: AI Agent
- **액션**:
  - AI 매칭 점수 기준 자동 필터링 (예: 70점 이상)
  - 우선순위 자동 정렬
- **결과**: 심사 대상 후보자 우선순위 리스트

---

### 5단계: 면접 일정 조율 및 진행

#### 5.1 면접 일정 조율
- **담당자**: HR 담당자
- **액션**:
  - 지원자에게 면접 일정 제안
  - 커뮤니케이션 자동화:
    - 이메일 템플릿 자동 생성
    - 면접 일정 안내 메시지 전송
    - 캘린더 연동 (향후 구현)
- **상태**: `INTERVIEW_SCHEDULED`

#### 5.2 면접 진행
- **담당자**: 면접관 (HR, 부서장, 팀장 등)
- **액션**:
  - 면접 평가표 작성
  - 면접 점수 입력
  - 면접 노트 작성
- **상태**: `INTERVIEW_SCHEDULED` → `INTERVIEW_COMPLETED`

---

### 6단계: 최종 평가 및 제안서 발행

#### 6.1 최종 평가
- **담당자**: 채용 담당자/부서장
- **액션**:
  - 서류 심사 점수 + 면접 점수 종합 평가
  - 최종 합격 여부 결정
- **결정**:
  - **합격**: 제안서 발행
  - **불합격**: 거절 메시지 전송

#### 6.2 제안서 생성 및 발행
- **담당자**: HR 담당자
- **액션**:
  - 제안서 생성 버튼 클릭
  - 제안서 내용 입력:
    - 포지션
    - 급여
    - 시작일
    - 복리후생
    - 기타 조건
  - AI 기반 제안서 템플릿 자동 생성
  - 지원자에게 이메일 자동 전송
- **상태**: `INTERVIEW_COMPLETED` → `OFFERED`

---

### 7단계: 제안서 수락 및 전자 서명

#### 7.1 제안서 수락
- **지원자**: 합격자
- **액션**:
  - 이메일에서 제안서 확인
  - 제안서 수락 버튼 클릭
  - 전자 서명 (디지털 서명)
- **상태**: `OFFERED` → `ACCEPTED`

#### 7.2 온보딩 준비
- **시스템**: 자동화
- **액션**:
  - 신규 직원 정보 자동 등록 (향후 구현)
  - 온보딩 체크리스트 생성
  - 입사 준비 안내 메시지 전송
- **상태**: `ACCEPTED` → 온보딩 프로세스 시작

---

## 🔄 상태 전이 다이어그램

```
[DRAFT] 
   ↓ (작성 완료)
[PENDING_APPROVAL]
   ↓ (승인)
[APPROVED]
   ↓ (배포)
[PUBLISHED]
   ↓ (지원 접수)
[APPLIED] → [SCREENING]
   ↓ (합격)        ↓ (불합격)
[INTERVIEW_SCHEDULED]  [REJECTED]
   ↓ (면접 완료)
[INTERVIEW_COMPLETED]
   ↓ (합격)
[OFFERED]
   ↓ (수락)
[ACCEPTED]
```

## 🤖 AI 자동화 기능

### 1. 채용 공고 자동 생성
- **입력**: 자격 요건, 직무 설명
- **출력**: 최적화된 채용 공고 문구
- **도구**: `RecruitmentAgent.generate_job_posting()`

### 2. 이력서 자동 파싱
- **입력**: 이력서 파일 (PDF, DOCX, TXT)
- **출력**: 구조화된 데이터 (JSON)
- **도구**: `ResumeParser.extract_text()` + AI 파싱

### 3. AI 매칭 점수 계산
- **입력**: 지원자 이력서 데이터 + 채용 공고 요구사항
- **출력**: 매칭 점수 (0-100) + 분석 리포트
- **도구**: `RecruitmentAgent.match_candidate()`

### 4. AI 소싱 (후보자 검색)
- **입력**: 채용 공고 설명 + 검색 조건
- **출력**: 적합한 후보자 목록
- **도구**: `AISourcing.search()`

### 5. 커뮤니케이션 자동화
- **이메일 템플릿**: 면접 안내, 합격/불합격 통보
- **자동 전송**: 상태 변경 시 자동 메시지 전송
- **도구**: `RecruitmentService.send_communication()`

## 📊 주요 지표 (KPI)

### 채용 효율성
- **평균 채용 기간**: 공고 게시 ~ 제안서 수락까지
- **지원자 대비 합격률**: 지원자 수 / 최종 합격자 수
- **면접 전환율**: 서류 합격 / 면접 합격

### AI 활용도
- **AI 매칭 평균 점수**: 모든 지원자의 평균 매칭 점수
- **AI 소싱 성공률**: AI 소싱으로 찾은 후보자 중 면접까지 진행된 비율
- **자동화율**: 자동 처리된 지원서 비율

### 품질 지표
- **이직률**: 입사 후 1년 이내 이직률
- **면접 만족도**: 면접관 평가 점수
- **온보딩 성공률**: 제안서 수락 후 실제 입사율

## 🔧 향후 개선 사항

### 단기 (1-3개월)
1. **캘린더 연동**: Google Calendar, Outlook 연동
2. **면접 평가표 템플릿**: 표준화된 평가 항목
3. **알림 시스템**: 실시간 채용 현황 알림
4. **대시보드 개선**: 채용 현황 시각화

### 중기 (3-6개월)
1. **비디오 면접**: 화상 면접 스케줄링 및 녹화
2. **레퍼런스 체크**: 자동 레퍼런스 확인
3. **배경 조사**: 자동 배경 조사 연동
4. **다국어 지원**: 글로벌 채용 지원

### 장기 (6-12개월)
1. **예측 분석**: AI 기반 이직률 예측
2. **다양성 분석**: 채용 다양성 지표 추적
3. **온보딩 자동화**: 입사 후 프로세스 자동화
4. **인재 풀 관리**: 과거 지원자 재활용

## 🚀 빠른 시작 가이드

### 1. 채용 공고 작성
```
1. 채용 관리 메뉴 클릭
2. "채용 공고 작성" 버튼 클릭
3. 필수 정보 입력 (제목, 부서, 직책 등)
4. 자격 요건 및 직무 설명 작성
5. "작성" 버튼 클릭
```

### 2. 채용 공고 승인 및 배포
```
1. 작성된 공고 목록에서 "승인" 버튼 클릭
2. 승인 완료 후 "배포" 버튼 클릭
3. 배포할 채용 사이트 선택
4. 배포 완료
```

### 3. 지원자 평가
```
1. 채용 공고 선택
2. "지원자 관리" 탭 클릭
3. 지원자 목록 확인
4. AI 매칭 점수 및 이력서 검토
5. "합격" 또는 "불합격" 버튼 클릭
6. 평가 점수 및 노트 입력
```

### 4. 제안서 발행
```
1. 면접 완료된 지원자 선택
2. "제안서" 버튼 클릭
3. 제안서 내용 입력 (급여, 시작일 등)
4. "발행" 버튼 클릭
5. 지원자에게 자동 전송
```
