# AI Agent Workflow 실행 가이드

## 개요

AI 어시스턴트는 단순히 질문에 답변하는 것을 넘어서, **실제 HR 시스템의 작업을 수행**할 수 있습니다.

## 지원하는 Workflow

### 1. 직원 정보 조회
**질문 예시:**
- "직원 ID 1번의 정보를 알려줘"
- "홍길동 직원의 정보를 조회해줘"
- "개발팀 직원 목록을 보여줘"

**실행 작업:**
- 데이터베이스에서 직원 정보 조회
- 부서별 필터링
- 상세 정보 제공

### 2. 대시보드 통계 조회
**질문 예시:**
- "현재 인사 현황을 알려줘"
- "이번 달 채용 현황은?"
- "전체 직원 수와 이직률을 보여줘"

**실행 작업:**
- 실시간 통계 데이터 조회
- 대시보드 정보 제공

### 3. 급여 정보 조회
**질문 예시:**
- "직원 1번의 급여 정보를 알려줘"
- "최근 급여 이력을 조회해줘"

**실행 작업:**
- 급여 계산 서비스 호출
- 급여 이력 조회

### 4. 근태 정보 조회
**질문 예시:**
- "직원 1번의 최근 30일 근태 현황을 알려줘"
- "이번 달 출근 현황을 보여줘"

**실행 작업:**
- 근태 관리 서비스 호출
- 근무 시간, 지각, 결근 통계 제공

### 5. 리포트 생성
**질문 예시:**
- "이직률 리포트를 만들어줘"
- "채용 지표 리포트를 생성해줘"
- "급여 비용 분석 리포트를 보여줘"

**실행 작업:**
- 리포트 서비스 호출
- 데이터 분석 및 리포트 생성

## 사용 예시

### 예시 1: 직원 정보 조회
```
사용자: "직원 ID 1번의 정보를 알려줘"

AI: [get_employee_info 도구 실행]
    직원 정보:
    - 이름: 홍길동
    - 사번: EMP001
    - 부서: 개발팀
    - 직책: 시니어 개발자
    - 입사일: 2023-01-15
    - 상태: 재직
    - 기본급: 5,000,000원
```

### 예시 2: 복합 질문
```
사용자: "개발팀 직원들의 근태 현황과 평균 급여를 알려줘"

AI: [list_employees 도구 실행]
    [get_attendance_summary 도구 실행]
    [calculate_payroll_info 도구 실행]
    
    개발팀 직원 현황:
    - 총 5명
    - 평균 근무 시간: 160시간/월
    - 평균 급여: 4,500,000원
    ...
```

## 기술 구현

### LangChain Agent 사용
- **Tools**: 실제 시스템 작업을 수행하는 도구들
- **Agent**: 도구를 선택하고 실행하는 AI
- **Executor**: Agent 실행 관리

### 도구 구조
```python
Tool(
    name="get_employee_info",
    func=get_employee_info,
    description="직원 정보 조회"
)
```

### Agent 실행 흐름
1. 사용자 질문 입력
2. Agent가 질문 분석
3. 필요한 도구 선택
4. 도구 실행 (실제 데이터 조회)
5. 결과를 바탕으로 답변 생성

## 제한사항

1. **읽기 전용 작업**: 현재는 조회 작업만 지원
2. **데이터베이스 접근**: DB 세션이 필요
3. **API 키 필요**: OpenAI API 키 필수

## 향후 개선 계획

- [ ] 급여 계산 실행
- [ ] 리포트 자동 생성 및 저장
- [ ] 복잡한 워크플로우 체인 실행
