# 모바일 반응형 가이드

## 개선 사항

### ✅ 완료된 모바일 최적화

1. **반응형 사이드바**
   - 모바일: 햄버거 메뉴로 변환
   - 데스크톱: 항상 표시
   - 오버레이로 모바일 메뉴 닫기

2. **반응형 그리드**
   - 모바일: 1열
   - 태블릿: 2열
   - 데스크톱: 4열

3. **반응형 테이블**
   - 모바일: 카드 형태로 표시
   - 데스크톱: 테이블 형태 유지

4. **반응형 텍스트 및 패딩**
   - 모바일: 작은 폰트, 작은 패딩
   - 데스크톱: 큰 폰트, 큰 패딩

5. **뷰포트 메타 태그**
   - 모바일에서 올바른 확대/축소 지원

## 반응형 브레이크포인트

Tailwind CSS의 기본 브레이크포인트:

- `sm`: 640px 이상 (작은 태블릿)
- `md`: 768px 이상 (태블릿)
- `lg`: 1024px 이상 (데스크톱)
- `xl`: 1280px 이상 (큰 데스크톱)

## 사용 예제

### 그리드 레이아웃
```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-4 gap-4 md:gap-6">
  {/* 모바일: 1열, 태블릿: 2열, 데스크톱: 4열 */}
</div>
```

### 텍스트 크기
```tsx
<h1 className="text-2xl md:text-3xl font-bold">
  {/* 모바일: 작은 크기, 데스크톱: 큰 크기 */}
</h1>
```

### 패딩
```tsx
<div className="p-4 md:p-6">
  {/* 모바일: 작은 패딩, 데스크톱: 큰 패딩 */}
</div>
```

### 조건부 표시
```tsx
{/* 모바일에서만 표시 */}
<div className="md:hidden">모바일 전용</div>

{/* 데스크톱에서만 표시 */}
<div className="hidden md:block">데스크톱 전용</div>
```

## 테스트 방법

1. **브라우저 개발자 도구**
   - F12 → 디바이스 툴바 (Ctrl+Shift+M)
   - 다양한 디바이스 크기 선택

2. **실제 디바이스**
   - 스마트폰에서 접속
   - 네트워크가 같은지 확인

3. **온라인 도구**
   - https://responsively.app/
   - 여러 디바이스 동시 미리보기
